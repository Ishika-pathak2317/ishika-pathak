# Ishika Pathak — Portfolio

Naruto-themed 3D premium portfolio built with vanilla HTML, CSS & JS.

## 🚀 Deploy to Vercel (3 steps)

### Option A — Drag & Drop (Easiest)
1. Go to **https://vercel.com** → Sign in with GitHub
2. Click **"Add New Project"** → **"Upload"**
3. Drag the entire `ishika-portfolio` folder → Deploy ✅

### Option B — GitHub (Recommended for updates)
1. Push this folder to a GitHub repo:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git push -u origin main
   ```
2. Go to **vercel.com** → "Add New Project" → Import from GitHub
3. Select the repo → **Deploy** ✅

### Option C — Vercel CLI
```bash
npm i -g vercel
cd ishika-portfolio
vercel
```
Follow the prompts — it deploys in ~30 seconds.
